HTML

1

Pengertian

HTML adalah bahasa markup yang digunakan untuk membuat struktur dan konten halaman web dengan menggunakan tag-tag. Ini memungkinkan pengguna untuk menambahkan teks, gambar, tautan, dan elemen-elemen lainnya ke halaman web. HTML bekerja bersama dengan CSS untuk desain tampilan dan JavaScript untuk interaksi dinamis.

2

Fungsi Html

HTML digunakan dalam pengembangan web untuk: Menentukan struktur halaman. Menambahkan konten seperti teks, gambar, dan tautan. Mengintegrasikan desain menggunakan CSS. Memperkaya interaksi dan fungsionalitas dengan JavaScript. Meningkatkan aksesibilitas dan optimasi mesin pencari.

3

Mengapa Harus HTML?

Belajar HTML penting karena: Membuat dasar pengembangan web. Memungkinkan penambahan konten seperti teks dan gambar. Mendukung desain halaman dengan CSS. Meningkatkan aksesibilitas dan optimasi mesin pencari. Memberi kemampuan untuk membangun halaman web yang fungsional.

Mengenal Tag Pada Html

Tag adalah sebuah penanda awalan dan akhiran dari sebuah elemen di HTML. Tag dibuat dengan kurung sudut (<...>), lalu di dalamnya berisi nama tag Tag selalu ditulis berpasangan. Ada tag pembuka dan ada tag penutupnya. Namun, ada juga beberapa tag yang tidak memiliki pasangan penutup. Tag penutup ditulis dengan menambahkan garis miring (/) di depan nama tag. Yang bergaris merah adalah tag pembuka sedangkan yang bergaris kuning adalah tag penutup. Setiap tag memiliki fungsi masing-masing. Ada yang digunakan untuk membuat judul, membuat link, membuat paragraf, heading, dan lain-lain.

9

Beberapa Tag Pada Html

Tag h1 - h6 digunakan untuk membuat heading pada artikel, sedangkan p untuk membuat paragraft

Mengenal Atribut

Atribut adalah kata kunci khusus yang berada di dalam tag pembuka. Atribut juga disebut sebagai modifier yang akan menentukan perliaku dari elemen. Atribut dapat ditambahkan pada elemen manapun. Ada juga elemen yang mewajibkan menggunakan atribut seperti elemen a, img, video, dll. href adalah atribut dari tag a, kalian bisa memberi link yang kalian inginkan untuk berpindah ke suatu laman website. Contoh seperti diatas, jika kalian klik pergi ke google maka akan mengarahkan ke google.com

CSS

1

Pengertian

CSS (Cascading Style Sheets) adalah bahasa yang digunakan untuk mendesain tampilan halaman web. Ini memungkinkan Anda mengontrol warna, layout, font, dan berbagai aspek visual lainnya dari elemen-elemen HTML dalam sebuah halaman web. Dengan menggunakan CSS, Anda dapat memisahkan struktur (HTML) dan tampilan (CSS) dari sebuah halaman web, memungkinkan pengembang untuk membuat halaman yang lebih menarik dan mudah dimodifikasi.

2

Fungsi Css Pada Html

CSS (Cascading Style Sheets) berfungsi untuk mengontrol tampilan dan gaya halaman web, memungkinkan pemisahan antara struktur konten (HTML) dan presentasi visualnya, membuat halaman responsif, menciptakan konsistensi gaya, dan mengoptimalkan kinerja situs web.

JavaScript

Materi Javascript Yang Disusun Oleh Raka Developer

JavaScript

1

Pengertian

JavaScript adalah bahasa pemrograman yang sering digunakan untuk membuat halaman web interaktif. Ini dapat digunakan untuk menambahkan fungsi seperti validasi formulir, animasi, manipulasi DOM, komunikasi dengan server, dan banyak lagi. JavaScript dapat dijalankan di browser pengguna tanpa perlu kompilasi, membuatnya sangat fleksibel dan berguna dalam pengembangan web.

2

Fungsi JavaScript Pada Html

JavaScript dalam pengembangan HTML berfungsi untuk: Menambahkan interaktivitas ke halaman web. Melakukan validasi formulir di sisi klien. Memanipulasi struktur dan konten halaman menggunakan DOM. Berkomunikasi dengan server untuk pengolahan data. Membuat animasi dan efek visual. Mengelola cookies dan penyimpanan lokal di browser.

3

Mengapa Harus JavaScript?

Belajar JavaScript penting karena: Memungkinkan Anda membuat halaman web interaktif. Dicari dalam industri teknologi. Memiliki ekosistem alat pengembangan yang kuat. Digunakan untuk pengembangan web dan server. Memberi ruang bagi kreativitas dan inovasi dalam pembangunan web.

5

Sejarah JavaScript

JavaScript diciptakan oleh Brendan Eich di Netscape pada tahun 1995. Awalnya, itu dikenal sebagai "LiveScript," tetapi kemudian diubah namanya menjadi "JavaScript" untuk memanfaatkan popularitas yang sedang meningkat dari bahasa pemrograman Java pada saat itu. Tujuan utama JavaScript adalah untuk menambahkan interaktivitas ke dalam halaman web. Pada awalnya, itu digunakan terutama untuk validasi formulir dan efek sederhana, tetapi seiring waktu, bahasa ini berkembang menjadi salah satu alat utama dalam pengembangan web, memungkinkan pembuatan aplikasi web yang semakin kompleks dan dinamis. JavaScript menjadi standar yang diadopsi oleh organisasi standar web seperti W3C. Kemudian, dengan penambahan fitur-fitur baru dan peningkatan kinerja yang signifikan, JavaScript menjadi bahasa yang sangat populer di web. Pada tahun 2009, Node.js, platform runtime JavaScript yang berjalan di server, diperkenalkan oleh Ryan Dahl, yang membuka pintu bagi pengembangan aplikasi web di luar browser. Selain itu, banyak kerangka kerja dan perpustakaan JavaScript, seperti jQuery, AngularJS, React, dan Vue.js, telah diciptakan untuk membantu pengembang dalam membangun aplikasi web yang lebih efisien dan kuat. Hingga saat ini, JavaScript tetap menjadi salah satu bahasa pemrograman paling penting dalam pengembangan web, dan terus berkembang dengan fitur-fitur baru dan peningkatan kinerja untuk memenuhi tuntutan yang semakin kompleks dalam pembangunan web modern.
